# OliVidi

Olive leaf disease detection using CNN image classification and AI-powered consultation reports.

Upload an olive leaf photo, get a disease prediction from a trained CNN model, and receive a detailed treatment report generated by a RAG agent (Ollama + ChromaDB).

## Features

- **OliNet CNN** - Binary classifier (healthy vs diseased) trained on olive leaf images
- **RAG Consultation Agent** - Retrieves scientific knowledge from ChromaDB and generates treatment reports via Ollama (Llama 3)
- **Streamlit Dashboard** - Upload images, view predictions, and read consultation reports
- **Offline Mode** - Works without Ollama using a built-in knowledge base fallback

## Dataset Reference

The model was trained and validated using the **[Olive (Zeytin) Dataset](https://www.kaggle.com/datasets/serhathoca/zeytin)** curated by **Serhat Kılıçarslan** on Kaggle. This dataset provides the foundational imagery required for classifying various states of olive health and pathogen presence

## Setup

### 1. Install dependencies

```bash
pip install -r requirements.txt
```

### 2. Install Ollama

Download from [ollama.com](https://ollama.com), then:

```bash
ollama pull llama3
```

> The app works without Ollama (offline fallback), but RAG-generated reports require it.

### 3. Prepare dataset

Place olive leaf images in:
- `dataset/healthy/` - healthy leaf images
- `dataset/diseased/` - diseased leaf images

### 4. Train the model

```bash
python data_pipeline.py
python olive_net.py
```

### 5. Run the app

```bash
streamlit run app.py
```

## Usage

1. Open the Streamlit app in your browser (default: `http://localhost:8501`)
2. Upload an olive leaf photo (JPG, PNG, BMP)
3. View the CNN prediction (healthy/diseased) with confidence score
4. Read the AI-generated consultation report with treatment recommendations

## Model Details

- **Architecture**: 3-block CNN (Conv2D + BatchNorm + MaxPool) with dense classifier
- **Input**: 224x224 RGB images
- **Output**: Binary (healthy=0, diseased=1)
- **Training**: 70/15/15 train/val/test split, EarlyStopping, ReduceLROnPlateau
